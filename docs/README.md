# 低代码平台文档

本目录包含低代码平台的核心文档，包括组件API、配置协议和实例示例。

## 目录结构

- `/components/`: 组件核心文档
  - `button.md`: 按钮组件文档
  - `input.md`: 输入框组件文档
  - `form-container.md`: 表单容器组件文档
- `/schemas/`: DSL 协议规范
  - `base-schema.md`: 基础 JSON 结构说明
  - `events.md`: 事件响应规范
- `/examples/`: 成功案例 (Few-shot 语料)
  - `login-page.json`: 登录页面的 JSON 配置示例
  - `search-list.md`: 描述：如何实现一个搜索列表

## 使用方式

这些文档可以通过多种方式使用：

1. **开发参考**：为开发者提供平台组件和配置规范的详细参考
2. **学习资料**：帮助新用户快速上手低代码平台
3. **RAG训练数据**：作为RAG系统的训练数据，为智能助手提供知识库

## 文档更新

文档内容应与平台功能保持同步，建议在以下情况下更新文档：

1. 添加新组件时
2. 修改现有组件API时
3. 变更配置协议时
4. 添加新的最佳实践示例时

## 文档格式

所有文档应遵循以下格式规范：

1. 使用Markdown格式
2. 代码示例应使用适当的语法高亮
3. 示例JSON应格式化以提高可读性
4. 包含适当的标题层次和目录
